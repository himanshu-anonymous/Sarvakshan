# @Sarvakshan/wwv-lib-incidents

Shared abstract base classes for Sarvakshan incident plugins (earthquakes, wildfires, conflicts)

## Usage
This package is part of the Sarvakshan plugin ecosystem. It provides the following capabilities:
- **ID:** `wwv-lib-incidents`
- **Category:** Miscellaneous
- **Format:** bundle

## Installation
Typically installed via the Sarvakshan Marketplace or discovered automatically.
If installing manually in a Next.js setup:
```bash
npm install @Sarvakshan/wwv-lib-incidents
```

## Architecture
This plugin adheres to the Sarvakshan standard plugin structure. As a `bundle` plugin, it connects to the core Event Bus and renders map capabilities.
